# NetMon: Network Monitoring System and Analysis

## üí° Overview

NetMon is a software that helps users to monitor network traffic when connected to their desired network. It is designed and developed for the use of any users, specifically network administrators to assist them in detecting any malicious activity in the network. This system is mainly designed for small-sized networks.

## ‚ú® Features

- **Packet Capture:** Capture packets like Wireshark.
- **Packet Analysis** View packet in details.
- **Host Discovery:** Discover hosts that are connected to the network.
- **Host Management:** Save host to database or disconnect host from network.
- **Host Security Check:** Simple port scan for host.
- **Basic Alert Detection:** New device, traffic spike, host scan, mac spoofing, port scan detection.
- **Traffic Visualisation:** Graph of number of packet(s) vs. time.
